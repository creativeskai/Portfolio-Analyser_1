# Quantfolio

NSE/BSE Portfolio Intelligence — educational, not SEBI-registered advice.

## Deploy to Vercel (5 minutes)

1. Upload this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Add environment variable: `REACT_APP_ANTHROPIC_API_KEY` = your Anthropic API key
4. Click Deploy

## Run locally

```bash
npm install
npm start
```

## Notes
- AI Chat tab requires an Anthropic API key
- All analysis is educational / illustrative only
